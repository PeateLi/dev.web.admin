### Tag 标签

<font face="黑体" color=#5E6D82 size=2>用于标记和选择。</font>

#### 用法
<font face="黑体" color=red size=2>由tags数组对象来选择tag的类型</font>

#### Tag Attributes
| 参数 | 说明 | 类型	 | 可选值 | 默认值 |
| :-----| :----: | :----: | :----: | :----: |
| tags | 标签组数据 | Array | — | [] |
| isClosableTag | 标签是否可关闭 | Boolean | — | true |

#### Tag Events
| 事件名称 | 说明 | 回调参数 |
| :-----| :----: | :----: |
| handleClose| 关闭 Tag 时触发的事件 | 被关闭的tag标签 |
