#### Button Attributes
| 参数 | 说明 | 类型	 | 可选值 | 默认值 |

| text | 按钮文字 | string | — | — |

| isDisabled | 是否禁用按钮 | boolean | — | false |
| isPlain | 是否朴素按钮 | boolean | — | false |
| isRound | 是否圆角按钮 | boolean | — | false |
| isCircle | 是否圆形按钮 | boolean | — | false |
| isIcon | 图标类名 | String | — | — |
| BottonSize | 尺寸 | String | medium / small / mini | — |
